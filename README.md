## Project Overview
This project is the **XRWVM Full Stack Developer Capstone** application.  
It demonstrates a complete full-stack dealership web platform built using:

- Django Backend
- REST APIs
- Microservices Integration
- Sentiment Analysis Service
- GitHub Actions CI/CD
- Cloud Deployment

The application allows users to:

- View car dealerships
- Filter dealerships by state
- View dealer details
- Submit and read reviews
- Perform sentiment analysis on reviews
- Authenticate users (Login / Logout)
